## 🚀 Установка исправленной версии

### Шаг 1: Отключите старую версию

```bash
cd /var/www/html/nextcloud
sudo -u www-data php occ app:disable one_c_web_client_v3
```

### Шаг 2: Удалите старую версию

```bash
# Удалите приложение
sudo -u www-data php occ app:remove one_c_web_client_v3

# Удалите файлы
rm -rf /var/www/html/nextcloud/apps/one_c_web_client_v3
```

### Шаг 3: Очистите кэш

```bash
sudo -u www-data php occ maintenance:repair
sudo -u www-data php occ cache:clear
```

### Шаг 4: Распакуйте исправленную версию

```bash
cd /var/www/html/nextcloud/apps
tar -xzf /tmp/one_c_web_client_v3_fixed.tar.gz
mv one_c_web_client_v3_clean one_c_web_client_v3
```

### Шаг 5: Установите права

```bash
chown -R www-data:www-data one_c_web_client_v3
chmod -R 755 one_c_web_client_v3
```

### Шаг 6: Установите приложение

```bash
cd /var/www/html/nextcloud
sudo -u www-data php occ app:install one_c_web_client_v3
```

### Шаг 7: Очистите кэш ещё раз

```bash
sudo -u www-data php occ maintenance:repair
sudo -u www-data php occ cache:clear
```

### Шаг 8: Перезапустите Apache

```bash
systemctl restart apache2
```
