# 🚀 УМНЫЙ ИНТЕРАКТИВНЫЙ УСТАНОВЩИК v4.0.0

**install_smart.sh** - полностью автоматизированный установщик с диагностикой

---

## ✨ ЧТО ДЕЛАЕТ СКРИПТ

### 1. ✅ Диагностика сервера
- Проверяет права root
- Ищет установку Nextcloud (автоматически или вручную)
- Определяет версию Nextcloud и PHP
- Проверяет статус Apache
- Ищет конфигурацию Apache
- Проверяет модули Apache (proxy, ssl, headers, и т.д.)
- Находит существующие настройки прокси для 1С

### 2. ✅ Поиск файлов установки
- Автоматически ищет архивы с приложением
- Проверяет валидность архива
- Позволяет указать путь вручную

### 3. ✅ Анализ конфигурации Apache
- Проверяет синтаксис Apache
- Анализирует VirtualHost конфигурацию
- Проверяет SSL настройки
- Определяет DocumentRoot

### 4. ✅ Настройка Apache (БЕЗ удаления!)
- **НЕ УДАЛЯЕТ существующие настройки!**
- Создаёт резервную копию
- Добавляет прокси ПЕРЕД закрывающим </VirtualHost>
- Проверяет синтаксис после изменений
- При ошибке предлагает восстановить резервную копию

### 5. ✅ Установка приложения
- Проверяет существующую установку
- Распаковывает архив
- Устанавливает права
- Устанавливает через occ
- Очищает кэш

### 6. ✅ Перезапуск и проверка
- Перезапускает Apache
- Проверяет статус приложения
- Проверяет синтаксис Apache
- Показывает URL для проверки

### 7. ✅ Финальный отчёт
- Показывает информацию об установке
- Сохраняет лог
- Показывает резервную копию
