# Обновление приложения интеграции 1С с Nextcloud

## Завершено: 10 февраля 2026

### Что было сделано:
1. Полностью переработано приложение интеграции 1С с Nextcloud
2. Добавлена поддержка нескольких серверов 1С
3. Реализована возможность добавления множества баз данных
4. Создан удобный интерфейс для администратора для управления серверами и базами
5. Улучшен пользовательский интерфейс с выбором базы для открытия во фрейме
6. Обновлено приложение до версии 2.0.0 и установлено на сервер

### Основные улучшения:
- Раздельное управление серверами и базами данных

- Возможность администратору добавлять/удалять сервера и базы данных
- Улучшенный интерфейс для пользователей с выбором базы и способом открытия
- Улучшенная система управления правами доступа
