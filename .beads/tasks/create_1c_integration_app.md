# Задача: Создание приложения интеграции 1С с Nextcloud

## Описание
Необходимо создать приложение интеграции 1С с Nextcloud, которое позволит пользователям получать доступ к различным базам 1С через веб-интерфейс Nextcloud. Приложение должно быть на русском языке и включать все необходимые компоненты.

## Требования
- Приложение должно быть полностью на русском языке
- Должна быть реализована система аутентификации и авторизации
- Должна быть возможность настройки подключения к серверу 1С
- Должна быть возможность выбора баз данных 1С
- Интерфейс должен быть интуитивно понятным
- Все кнопки и функции должны быть активны и работать корректно

## Компоненты
- Контроллеры (Page, Config, Access)
- Сервисы (OneCConnector, AuthService)
- Шаблоны (index, admin-settings, iframe)
- Файлы конфигурации (info.xml, routes.php)
- Классы настроек (AdminSettings, AdminSection)

## План
1. Создать структуру приложения
2. Реализовать все компоненты
3. Обеспечить русский язык интерфейса
4. Установить приложение в Nextcloud
5. Проверить работу всех функций
