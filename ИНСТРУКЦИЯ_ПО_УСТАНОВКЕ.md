## 📋 ИНСТРУКЦИЯ ПО УСТАНОВКЕ

### Шаг 1: Подключитесь к серверу

```bash
ssh smidt@10.1.72.70
# Пароль: Apple0589
```

### Шаг 2: Распакуйте пакет

```bash
cd /tmp
tar -xzf one_c_web_client_v3_package.tar.gz
cd install_package
```

### Шаг 3: Запустите установщик

```bash
chmod +x install.sh
sudo ./install.sh
```

### Шаг 4: Следуйте инструкциям скрипта

Скрипт автоматически:
1. ✅ Проверит права root
2. ✅ Найдёт Nextcloud (`/var/www/nextcloud`)
3. ✅ Найдёт конфиг Apache
4. ✅ Проверит модули Apache
5. ✅ Распакует приложение
6. ✅ Установит через occ
7. ✅ Настроит прокси (если базы 1С уже добавлены)
8. ✅ Проверит работу
9. ✅ Создаст резервную копию

## ✅ ПРОВЕРКА ПОСЛЕ УСТАНОВКИ

### 1. Проверьте приложение

```bash
sudo -u www-data php /var/www/nextcloud/occ app:list | grep one_c
```

Должно быть:
```
one_c_web_client_v3: 6.0.0
```

### 2. Проверьте Apache

```bash
sudo apache2ctl configtest
```

Должно быть: `Syntax OK`

### 3. Откройте в браузере

**Админка:**
```
https://cloud.smidt.keenetic.pro/index.php/settings/admin/one_c_web_client_v3
```

**Клиентская часть:**
```
https://cloud.smidt.keenetic.pro/index.php/apps/one_c_web_client_v3/
```
