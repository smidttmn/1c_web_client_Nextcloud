## 🎯 Ключевые особенности v10.0.0

### ✅ Автоматическая настройка прокси

- **Определяет тип конфигурации:**
  - SSL (порт 443) - `nextcloud-le-ssl.conf`
  - NON_SSL (порт 80) - `nextcloud.conf`

- **Настраивает прокси ПРАВИЛЬНО:**
  - `ProxyPass` **ДО** всех исключений
  - `ProxyPassMatch` для путей
  - `mod_substitute` для переписывания URL
  - `AllowOverride None` для работы прокси

- **Не ломает существующие настройки:**
  - SSL сертификаты остаются нетронутыми
  - Резервная копия создаётся автоматически

## 🆘 Устранение проблем

### Ошибка 404 при открытии 1С

**Проблема:** ProxyPath добавлен после исключений

**Решение:**
```bash
cd /home/smidt/nc1c
sudo ./setup_apache_proxy_auto.sh
```

### Ошибка 502 Bad Gateway

**Проблема:** Сервер 1С недоступен

**Решение:**
```bash
# Проверьте доступность сервера 1С
curl -k https://10.72.1.5/sgtbuh/

# Проверьте логи Apache
sudo tail -f /var/log/apache2/error.log
```

### Mixed Content (блокировка браузером)

**Проблема:** Браузер блокирует HTTP ресурсы на HTTPS странице

**Решение:**
1. Откройте настройки браузера
2. Найдите "Небезопасный контент" или "Mixed Content"
3. Разрешите для `cloud.smidt.keenetic.pro`
4. ИЛИ настройте HTTPS на сервере 1С

### 1С не загружается полностью

**Проблема:** mod_substitute не включён

**Решение:**
```bash
sudo a2enmod substitute
sudo systemctl restart apache2
```
