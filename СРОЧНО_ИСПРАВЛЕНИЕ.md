# 🚨 СРОЧНО! Исправление ошибки 500

## ⚡ БЫСТРОЕ РЕШЕНИЕ

**На сервере drive.nppsgt.com выполните:**

```bash
# 1. Отключите приложение
cd /var/www/html/nextcloud
sudo -u www-data php occ app:disable one_c_web_client_v3

# 2. Очистите кэш
sudo -u www-data php occ maintenance:repair
sudo -u www-data php occ cache:clear

# 3. Проверьте работу
# Откройте: https://drive.nppsgt.com/index.php/settings/admin
```

---

## 🔍 Если не помогло - удалите приложение

```bash
# 1. Отключите
sudo -u www-data php occ app:disable one_c_web_client_v3

# 2. Удалите
sudo -u www-data php occ app:remove one_c_web_client_v3

# 3. Удалите файлы
rm -rf /var/www/html/nextcloud/apps/one_c_web_client_v3

# 4. Очистите кэш
sudo -u www-data php occ maintenance:repair
sudo -u www-data php occ cache:clear

# 5. Перезапустите Apache
systemctl restart apache2
```

---

## 📋 Проверьте логи

```bash
# Логи Nextcloud
sudo -u www-data php occ log:read | tail -50

# Логи Apache
tail -100 /var/log/apache2/nextcloud/nextcloud_error.log
```

---

## ✅ Проверка

После исправления проверьте:

```bash
# Статус Nextcloud
sudo -u www-data php occ status

# Статус приложений
sudo -u www-data php occ app:list --enabled
```

**Откройте в браузере:**
- https://drive.nppsgt.com/index.php/settings/admin
