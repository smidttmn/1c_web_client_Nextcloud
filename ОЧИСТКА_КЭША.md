# 🔄 ОЧИСТКА КЭША ДЛЯ one_c_web_client_v3

**Проблема:** Браузер использует старый index.js из кэша

---

## 🚀 РЕШЕНИЕ

### Шаг 1: Очистите кэш браузера

**Chrome/Edge:**
1. Нажмите `Ctrl+Shift+Delete`
2. Выберите "Кэшированные изображения и файлы"
3. Нажмите "Удалить"

**Firefox:**
1. Нажмите `Ctrl+Shift+Delete`
2. Выберите "Кэш"
3. Нажмите "OK"

**ИЛИ:**
- Откройте страницу в режиме инкогнито (`Ctrl+Shift+N` или `Ctrl+Shift+P`)

---

### Шаг 2: Очистите кэш Nextcloud

```bash
# На сервере cloud.smidt.keenetic.pro:

sudo -u www-data php /var/www/nextcloud/occ maintenance:repair
sudo -u www-data php /var/www/nextcloud/occ memcache:clear 2>/dev/null || true
sudo systemctl restart apache2
```

---

### Шаг 3: Проверьте версию index.js

**Откройте консоль разработчика (F12)** и выполните:

```javascript
console.log('one_c_web_client_v3: Checking proxy...');
```

**ИЛИ посмотрите в Network (F12 → Network):**

1. Откройте: `https://cloud.smidt.keenetic.pro/index.php/apps/one_c_web_client_v3/`
2. Найдите файл `index.js`
3. Посмотрите содержимое

**Должно быть:**
```javascript
// Извлекаем путь из URL
const urlPath = new URL(url).pathname;

// Открываем через прокси Nextcloud
const proxyPath = '/one_c_web_client_v3' + urlPath;

console.log('one_c_web_client_v3: Opening via proxy:', proxyPath);
```

**Если старый код:**
```javascript
// Старый код (неправильно)
openDatabase(url, dbName);
```

---

### Шаг 4: Принудительная перезагрузка

**Chrome/Edge:**
- Нажмите `Ctrl+Shift+R` (или `Cmd+Shift+R` на Mac)

**Firefox:**
- Нажмите `Ctrl+F5`

---

## ✅ ПРОВЕРКА

**1. Откройте консоль (F12)**

**2. Нажмите на кнопку базы 1С**

**3. В консоли должно быть:**
```
one_c_web_client_v3: Opening database: {url: 'https://10.72.1.5/sgtbuh', dbName: 'CUN BUH'}
one_c_web_client_v3: Opening via proxy: /one_c_web_client_v3/sgtbuh
```

**4. В Network (F12 → Network) должно быть:**
```
GET /one_c_web_client_v3/sgtbuh 200 OK
```

**А НЕ:**
```
GET https://10.72.1.5/sgtbuh 499
```

## 🔧 ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ

### Проверьте файл на сервере

```bash
# На сервере:
cat /var/www/nextcloud/apps/one_c_web_client_v3/js/index.js | grep -A5 "proxyPath"
```

**Должно быть:**
```javascript
const proxyPath = '/one_c_web_client_v3' + urlPath;
console.log('one_c_web_client_v3: Opening via proxy:', proxyPath);
openDatabase(proxyPath, dbName);
```

### Переустановите приложение

```bash
# Удалите
sudo -u www-data php /var/www/nextcloud/occ app:disable one_c_web_client_v3
sudo -u www-data php /var/www/nextcloud/occ app:remove one_c_web_client_v3
sudo rm -rf /var/www/nextcloud/apps/one_c_web_client_v3

# Распакуйте заново
cd /tmp/install_package
sudo ./install.sh
```
