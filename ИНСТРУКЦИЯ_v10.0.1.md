## ✅ ЧТО ИСПРАВЛЕНО В v10.0.1:

**Проблема v10.0.0:**
```apache
# ОШИБКА: нет слэша перед путём
ProxyPass sgtbuh https://10.72.1.5sgtbuh
```

**Исправление v10.0.1:**
```apache
# ПРАВИЛЬНО: слэш добавлен
ProxyPass /sgtbuh https://10.72.1.5/sgtbuh
```

**Результат:** 1С открывается без ошибки 404 ✅
