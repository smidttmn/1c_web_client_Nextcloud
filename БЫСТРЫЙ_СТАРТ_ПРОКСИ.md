# ⚡ БЫСТРЫЙ СТАРТ - НАСТРОЙКА ПРОКСИ ЗА 5 МИНУТ

## Проблема решена!

Теперь прокси настраивается **автоматически** при установке приложения.

---

## 🚀 Вариант 1: Полная установка с нуля

### Одна команда:

```bash
sudo /path/to/install_package/install.sh
```

### Что произойдёт:

1. Установка приложения
2. Настройка серверов 1С (интерактивно)
3. **Автоматическая настройка прокси**
4. Перезапуск Apache

**Итог:** 1С работает сразу после установки!

---

## 🚀 Вариант 2: Быстрая настройка прокси

Если приложение уже установлено:

```bash
sudo /path/to/setup_apache_proxy_auto.sh
```

### Что произойдёт:

1. Определение конфигурации (SSL/NON_SSL)
2. Получение сервера 1С из Nextcloud
3. Настройка прокси
4. Перезапуск Apache

**Итог:** Прокси настроен за 1 минуту!

## 🆘 Если не работает

### Ошибка 404:

```bash
# Проверьте, что ProxyPass добавлен ДО исключений
sudo grep -B 5 "ProxyPass /core !" /etc/apache2/sites-available/nextcloud-le-ssl.conf

# Должно быть:
# ProxyPass /one_c_web_client_v3 ...
# ProxyPass /core !
```

### Ошибка 502:

```bash
# Проверьте доступность сервера 1С
curl -k https://10.72.1.5/sgtbuh/
```

### Mixed Content:

- Разрешите небезопасный контент в браузере
- Или настройте HTTPS на сервере 1С
