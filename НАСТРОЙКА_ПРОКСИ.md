# 🚀 НАСТРОЙКА ПРОКСИ ДЛЯ 1С

## Проблема
При открытии базы 1С ошибка 499 - Apache не настроен для проксирования запросов.

## Решение

### Выполните на сервере drive.nppsgt.com:

```bash
# 1. Подключитесь по SSH
ssh root@drive.nppsgt.com

# 2. Запустите интерактивный установщик
sudo /root/install_interactive_v5.sh
```

### Скрипт автоматически:
1. ✅ Найдёт Nextcloud
2. ✅ Найдёт конфиг Apache
3. ✅ Проверит модули Apache
4. ✅ Получит список баз 1С из Nextcloud
5. ✅ Добавит ProxyPass для каждой базы
6. ✅ Создаст резервную копию
7. ✅ Проверит синтаксис Apache
8. ✅ Перезапустит Apache
9. ✅ Проверит работу после установки

### После установки:
1. Обновите страницу в браузере (Ctrl+Shift+R)
2. Нажмите на кнопку базы 1С
3. 1С должна открыться через прокси Nextcloud

---

## Проверка вручную

Если скрипт не работает, проверьте вручную:

```bash
# 1. Проверьте конфиг Apache
sudo grep -A10 "one_c_web_client" /etc/apache2/sites-available/nextcloud.conf

# Должно быть что-то вроде:
# ProxyPass /one_c_web_client_v3 https://10.72.1.5/one_c_web_client_v3

# 2. Проверьте синтаксис Apache
sudo apache2ctl configtest

# 3. Перезапустите Apache
sudo systemctl restart apache2
```
