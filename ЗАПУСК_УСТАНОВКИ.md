# 🚀 ЗАПУСК УСТАНОВКИ (обновлённый скрипт)

**Проблема:** Команда `cache:clear` недоступна в вашей версии Nextcloud

**Решение:** Скрипт обновлён, используйте `memcache:clear` вместо `cache:clear`

---

## ⚡ ЗАПУСТИТЕ ЗАНОВО

**На сервере drive.nppsgt.com выполните:**

```bash
cd /tmp
sudo ./install_fixed.sh
```

**ВСЁ!** Обновлённый скрипт сам:
- ✅ Отключит старую версию
- ✅ Удалит файлы
- ✅ Очистит кэш (правильной командой)
- ✅ Распакует исправленную версию
- ✅ Установит приложение
- ✅ Перезапустит Apache

---

## 📋 Если ошибка осталась

### Проверьте версию Nextcloud

```bash
sudo -u www-data php occ status
```

### Проверьте доступные команды кэша

```bash
sudo -u www-data php occ list | grep memcache
```

Должно быть:
```
memcache
memcache:clear
memcache:distributed
```

### Проверьте логи

```bash
# Логи Nextcloud
sudo -u www-data php occ log:read | tail -50

# Логи Apache
tail -100 /var/log/apache2/nextcloud/nextcloud_error.log
```

## 🔧 Ручная установка (если скрипт не работает)

```bash
# 1. Отключите приложение
cd /var/www/html/nextcloud
sudo -u www-data php occ app:disable one_c_web_client_v3

# 2. Удалите приложение
sudo -u www-data php occ app:remove one_c_web_client_v3

# 3. Удалите файлы
rm -rf /var/www/html/nextcloud/apps/one_c_web_client_v3

# 4. Очистите кэш
sudo -u www-data php occ maintenance:repair
sudo -u www-data php occ memcache:clear 2>/dev/null || true

# 5. Распакуйте новую версию
cd /var/www/html/nextcloud/apps
tar -xzf /tmp/one_c_web_client_v3_fixed.tar.gz
mv one_c_web_client_v3_clean one_c_web_client_v3

# 6. Установите права
chown -R www-data:www-data one_c_web_client_v3
chmod -R 755 one_c_web_client_v3

# 7. Установите приложение
cd /var/www/html/nextcloud
sudo -u www-data php occ app:install one_c_web_client_v3

# 8. Очистите кэш
sudo -u www-data php occ maintenance:repair
sudo -u www-data php occ memcache:clear 2>/dev/null || true

# 9. Перезапустите Apache
systemctl restart apache2
```
