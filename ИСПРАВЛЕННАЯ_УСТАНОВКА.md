## 🔧 Как работает исправленный скрипт

1. **Ищет архив с приложением:**
   - `./one_c_web_client_v3_deploy.tar.gz`
   - `./one_c_web_client_v3_full.tar.gz`
   - `/tmp/one_c_web_client_v3_*.tar.gz`

2. **Если архив найден:**
   - Распаковывает во временную директорию
   - Находит приложение по `appinfo/info.xml`
   - Копирует в `apps/one_c_web_client_v3`

3. **Если архив не найден:**
   - Ищет директорию с приложением
   - Копирует оттуда

4. **Если ничего не найдено:**
   - Выводит ошибку с инструкцией


### Ошибка: "Конфиг Apache не найден"

**Решение:**
```bash
# Найдите конфиг:
find /etc/apache2 -name "*.conf" | grep -i nextcloud

# Или создайте новый:
sudo nano /etc/apache2/sites-available/nextcloud.conf
```
