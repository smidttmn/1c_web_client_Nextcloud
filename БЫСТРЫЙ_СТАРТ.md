# 🚀 БЫСТРЫЙ СТАРТ - Установка на продакшн

**Дата:** 13 марта 2026  
**Версия:** 3.2.1

---

## ⚡ Автоматическая установка (1 команда!)

```bash
cd /tmp && ./install_production.sh
```

**ВСЁ!** Скрипт сам:
- ✅ Найдёт архив в `/tmp/`
- ✅ Скопирует приложение в Nextcloud
- ✅ Настроит Apache
- ✅ Установит приложение
- ✅ Перезапустит Apache

---

## 📋 Пошаговая установка

### Шаг 1: Скопируйте файлы в /tmp

```bash
# С локального компьютера
scp install_production.sh root@drive.technoorganic.info:/tmp/
scp one_c_web_client_v3_nc30_deploy.tar.gz root@drive.technoorganic.info:/tmp/

# ИЛИ используйте файлы из /home/smidt/
cp /home/smidt/install_production.sh /tmp/
cp /home/smidt/one_c_web_client_v3_deploy.tar.gz /tmp/
```

### Шаг 2: Запустите установку

```bash
cd /tmp
chmod +x install_production.sh
./install_production.sh
```

### Шаг 3: Проверьте установку

```bash
# Проверка приложения
sudo -u www-data php occ app:list | grep one_c

# Должно быть: one_c_web_client_v3: 3.2.1
```

---

## 🔧 Если ошибка "Архив не найден"

**Проблема:** Скрипт не находит архив

**Решение:**
```bash
# Скопируйте архив в /tmp
cp /home/smidt/one_c_web_client_v3_deploy.tar.gz /tmp/

# ИЛИ укажите путь к архиву при запуске
cd /tmp
./install_production.sh /home/smidt/one_c_web_client_v3_deploy.tar.gz
```

## 🐛 Решение проблем

### Ошибка 500

Смотрите: `FIX_ERROR_500_NC30.md`

### Ошибка "Архив не найден"

```bash
# Проверьте наличие архива
ls -lh /tmp/one_c_web_client_v3*.tar.gz

# Если нет - скопируйте
cp /home/smidt/one_c_web_client_v3_deploy.tar.gz /tmp/
```

### Ошибка "Nextcloud не найден"

```bash
# Проверьте путь
ls -la /var/www/html/nextcloud/occ

# Если нет - найдите Nextcloud
find /var/www -name occ 2>/dev/null
```
