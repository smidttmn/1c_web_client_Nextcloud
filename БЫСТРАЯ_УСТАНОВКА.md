## 🚀 Установка с нуля (3 команды)

### Шаг 1: Очистка (если нужно)
```bash
cd /home/smidt/nc1c
sudo ./reset_nextcloud.sh
```

### Шаг 2: Установка
```bash
cd /home/smidt/nc1c
sudo ./install_package/install.sh
```

### Шаг 3: Проверка
```bash
# Откройте в браузере
https://cloud.smidt.keenetic.pro
# Нажмите Ctrl+Shift+R и найдите иконку 1С
```

---

## 📋 Что спросит установщик:

1. **Продолжить установку?** → `Y`
2. **Добавить сервер 1С сейчас?** → `Y`
3. **Название базы:** → например, `БУХ СГТ`
4. **URL сервера 1С:** → например, `https://10.72.1.5/sgtbuh`
5. **Добавить ещё один?** → `N` (если одна база)

## 🆘 Если не работает:

### Ошибка 404:
```bash
sudo ./setup_apache_proxy_auto.sh
```

### Ошибка 502:
```bash
# Проверьте доступность 1С
curl -k https://10.72.1.5/sgtbuh/
```

### Mixed Content:
- Разрешите небезопасный контент в браузере
- Или настройте HTTPS на 1С
