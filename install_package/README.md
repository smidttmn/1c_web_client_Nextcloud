## 🚀 Быстрая установка

### 1. Скопируйте пакет на сервер

```bash
scp -r one_c_web_client_v3_package root@your-server:/tmp/
```

### 2. Запустите установку

```bash
cd /tmp/one_c_web_client_v3_package
chmod +x install.sh
sudo ./install.sh
```

### 3. Следуйте инструкциям

Скрипт автоматически:
- ✅ Найдёт Nextcloud (автоматически)
- ✅ Найдёт конфигурацию Apache (**SSL или NON_SSL**)
- ✅ Проверит модули Apache (включит отсутствующие)
- ✅ Установит приложение
- ✅ **Интерактивно запросит серверы 1С**
- ✅ **АВТОМАТИЧЕСКИ настроит прокси** (ПРАВИЛЬНО!)
- ✅ Проверит синтаксис Apache
- ✅ Перезапустит Apache
- ✅ Создаст резервную копию

---

## 🎯 ГЛАВНОЕ: Прокси настраивается АВТОМАТИЧЕСКИ!

### Что делает установщик:

1. **Определяет тип конфигурации:**
   - SSL (порт 443) - `nextcloud-le-ssl.conf`
   - NON_SSL (порт 80) - `nextcloud.conf`

2. **Настраивает прокси ПРАВИЛЬНО:**
   - `ProxyPass` **ДО** всех исключений
   - `ProxyPassMatch` для путей
   - `mod_substitute` для переписывания URL
   - `AllowOverride None` для работы прокси

3. **Не ломает существующие настройки:**
   - SSL сертификаты остаются нетронутыми
   - Резервная копия создаётся автоматически

### Результат:

**Сразу после установки 1С работает через прокси!** 🎉

## 🔧 Отдельная настройка прокси

Если приложение уже установлено, но прокси не настроен:

```bash
# Скопируйте скрипт настройки
scp setup_apache_proxy_auto.sh root@your-server:/tmp/

# Запустите настройку
cd /tmp
chmod +x setup_apache_proxy_auto.sh
sudo ./setup_apache_proxy_auto.sh
```

Скрипт:
- Автоматически определит тип конфигурации (SSL/NON_SSL)
- Получит сервер 1С из Nextcloud (или спросит вручную)
- Настроит прокси ПРАВИЛЬНО
- Перезапустит Apache

## 🐛 Решение проблем

### Ошибка: "Nextcloud не найден"

Укажите путь вручную:
```bash
# Скрипт спросит путь, введите например:
/var/www/nextcloud
```

### Ошибка: "Архив приложения не найден"

Проверьте структуру пакета:
```bash
ls -la app/
# Должен быть файл: one_c_web_client_v3.tar.gz
```

### Ошибка: "Apache syntax error"

Скрипт автоматически восстановит резервную копию. Лог установки:
```
/tmp/one_c_install_ГГГГММДД_ЧЧММСС.log
```

## 📞 Логи

Лог установки сохраняется в:

Отправьте этот лог разработчику если что-то не работает.
