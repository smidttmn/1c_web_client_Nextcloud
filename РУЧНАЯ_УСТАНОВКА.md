## 📦 Шаг 2: Установите приложение в Nextcloud

```bash
# Скопируйте в Nextcloud
sudo cp -r one_c_web_client_v3_clean /var/www/html/nextcloud/apps/one_c_web_client_v3

# Установите права
sudo chown -R www-data:www-data /var/www/html/nextcloud/apps/one_c_web_client_v3
sudo chmod -R 755 /var/www/html/nextcloud/apps/one_c_web_client_v3

# Установите приложение
sudo -u www-data php /var/www/html/nextcloud/occ app:install one_c_web_client_v3

# Очистите кэш
sudo -u www-data php /var/www/html/nextcloud/occ maintenance:repair
```

## 🔧 Шаг 3: Настройте Apache

### Откройте конфиг

```bash
sudo nano /etc/apache2/sites-available/nextcloud.conf
```

### Найдите секцию `<VirtualHost *:443>`

Ищите строки:
```apache
<VirtualHost *:443>
   ServerName localhost
   ServerAlias drive.nppsgt.com
   DocumentRoot /var/www/html/nextcloud
   ...
   SSLCertificateFile /etc/letsencrypt/live/drive.nppsgt.com-0002/fullchain.pem
   SSLCertificateKeyFile /etc/letsencrypt/live/drive.nppsgt.com-0002/privkey.pem
```

### Добавьте ПЕРЕД закрывающим `</VirtualHost>`

**В конец HTTPS VirtualHost**, перед `</VirtualHost>` добавьте:

```apache
    # one_c_web_client_v3 - Прокси для 1С
    SSLProxyEngine on
    SSLProxyVerify none
    SSLProxyCheckPeerCN off
    SSLProxyCheckPeerName off

    # Исключения для статических файлов Nextcloud (ОБЯЗАТЕЛЬНО ДО ProxyPass!)
    ProxyPass /core !
    ProxyPass /apps !
    ProxyPass /dist !
    ProxyPass /js !
    ProxyPass /css !
    ProxyPass /l10n !
    ProxyPass /index.php !
    ProxyPass /loleaflet !
    ProxyPass /browser !
    ProxyPass /hosting !
    ProxyPass /cool !

    # Прокси для 1С сервера: https://10.72.1.5
    ProxyPass /one_c_web_client_v3 https://10.72.1.5/one_c_web_client_v3 retry=0 timeout=60
    ProxyPassReverse /one_c_web_client_v3 https://10.72.1.5/one_c_web_client_v3
    ProxyPassReverseCookiePath / /

    # Разрешение фреймов и CSP
    Header unset X-Frame-Options
    Header always set Content-Security-Policy "frame-ancestors 'self'; frame-src *; connect-src *; script-src 'self' 'unsafe-inline' 'unsafe-eval' *; style-src 'self' 'unsafe-inline' *;"
```

## ⚠️ Возможные проблемы

### 1. Ошибка: "Syntax error"

**Причина:** Неправильно добавлены директивы в конфиг

**Решение:**
```bash
# Проверьте, что директивы добавлены В КОНЕЦ VirtualHost *:443
# Перед закрывающим </VirtualHost>
sudo apache2ctl configtest
```

### 2. Ошибка: "ProxyPass conflicts with existing path"

**Причина:** ProxyPass добавлен ДО исключений

**Решение:**
```apache
# Исключения ДОЛЖНЫ быть ДО ProxyPass!
ProxyPass /core !
ProxyPass /apps !
ProxyPass /js !
ProxyPass /css !

# Потом ProxyPass
ProxyPass /one_c_web_client_v3 ...
```

### 3. Ошибка: "SSL Proxy Error"

**Причина:** Не включен SSLProxyEngine

**Решение:**
```apache
SSLProxyEngine on
SSLProxyVerify none
SSLProxyCheckPeerCN off
SSLProxyCheckPeerName off
```

### 4. CSP блокировка

**Причина:** Content-Security-Policy блокирует фреймы

**Решение:**
```apache
Header always set Content-Security-Policy "frame-ancestors 'self'; frame-src *; connect-src *; script-src 'self' 'unsafe-inline' 'unsafe-eval' *; style-src 'self' 'unsafe-inline' *;"
```

### 5. Mixed Content

**Причина:** Nextcloud на HTTPS, 1С на HTTP

**Решение:**
- Настройте HTTPS на 1С сервере
- Или используйте ProxyPass с HTTPS
