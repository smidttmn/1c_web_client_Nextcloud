## 🐛 НАЙДЕНЫ И ИСПРАВЛЕНЫ ПРОБЛЕМЫ

### 1. Неправильное имя приложения в коде

**Проблема:** Везде использовалось `'one_c_web_client'` вместо `'one_c_web_client_v3'`

**Исправлено в файлах:**
- ✅ `lib/Settings/AdminSettings.php`
- ✅ `lib/Controller/PageController.php`
- ✅ `lib/Controller/ConfigController.php`
- ✅ `appinfo/info.xml` (routes)
- ✅ `lib/AppInfo/Application.php` (APP_ID)

**Изменения:**
```php
// Было
Util::addScript('one_c_web_client', 'admin_settings');
$this->config->getAppValue('one_c_web_client', 'databases', '[]');

// Стало
Util::addScript('one_c_web_client_v3', 'admin_settings');
$this->config->getAppValue('one_c_web_client_v3', 'databases', '[]');
```

---

### 2. Несовместимость с Nextcloud 33

**Проблема:** В info.xml указано `max-version="32"`, а на сервере NC 33

**Исправлено:**
```xml
<!-- Было -->
<nextcloud min-version="30" max-version="32"/>

<!-- Стало -->
<nextcloud min-version="33" max-version="99"/>
```

---

### 3. Несовместимость с PHP 8.4

**Проблема:** В info.xml указано `max-version="8.3"`, а на сервере PHP 8.4.18

**Исправлено:**
```xml
<!-- Было -->
<php min-version="8.0" max-version="8.3"/>

<!-- Стало -->
<php min-version="8.0" max-version="8.4"/>
```

---

### 4. Неправильный путь к Nextcloud

**Проблема:** Скрипты использовали `/var/www/html/nextcloud`

**Исправлено:** Скрипты используют `/var/www/nextcloud`

---

### 5. Команда memcache:clear не существует

**Проблема:** В Nextcloud 33 нет команды `memcache:clear`

**Исправлено:** Используется `maintenance:repair` как fallback

## 🚀 УСТАНОВКА НА НОВЫЙ СЕРВЕР

**Автоматическая установка:**

```bash
# Скопируйте скрипт на сервер
scp auto_install.sh user@server:/tmp/

# Запустите установку
ssh user@server
cd /tmp
chmod +x auto_install.sh
sudo ./auto_install.sh
```

**Ручная установка:**

```bash
# 1. Скачайте приложение
cd /var/www/nextcloud/apps
git clone -b feature/proxy-with-rewrite https://github.com/smidttmn/one_c_web_client.git
cd one_c_web_client

# 2. Скопируйте приложение
cp -r one_c_web_client_v3_clean one_c_web_client_v3
chown -R www-data:www-data one_c_web_client_v3
chmod -R 755 one_c_web_client_v3

# 3. Включите приложение
sudo -u www-data php occ app:install one_c_web_client_v3
sudo -u www-data php occ maintenance:repair
```
